# 🤖 Code Companion Pro

An advanced, AI-integrated IDE companion built for mobile-first development. Featuring a tri-panel workspace and multi-language support.

## 🚀 Key Components
- **WorkspaceLayout:** A resizable 3-pane system (Editor | Preview | Chat).
- **AI Chat Sidebar:** Real-time development assistance with Framer Motion animations.
- **Language Switcher:** Support for JS, Python, Java, and C++.
- **Action Bar:** Quick-access tools for running code and file management.

## 🛠️ Setup
1. **Install Dependencies:** `npm install`
2. **Run Dev Server:** `npm run dev`
3. **Build for Production:** `npm run build`

## 📂 Architecture
- `src/components`: UI components (Shadcn + Custom)
- `src/lib`: Core logic and language definitions
- `public`: Static assets and PWA manifest
